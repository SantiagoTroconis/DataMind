In handleUndo (around line 303-306), the current success block reads only data.data:

```typescript
const data = await response.json();
if (data.status === 'success' && data.data) {
  setGridData(data.data);
}
```

Extend it to also sync chart state from the undo response fields has_chart and chart_data:

```typescript
const data = await response.json();
if (data.status === 'success' && data.data) {
  setGridData(data.data);
  if (!data.has_chart) {
    setChartData(null);
  } else if (data.chart_data) {
    setChartData(data.chart_data);
  }
}
```

No other changes. Do not touch the catch block, the finally block, or any other part of the file.
